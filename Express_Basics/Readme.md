# Setup express js
```
   
   1. npm init -y
   2. npm i express (inside folder)

   3. create file express.js
   const express = require("express");
   const app = express();

   app.listen(3000, ()=>{
    console.log("First Sever created and started at port 3000");
   });

   node express.js 
   generalised => node {file name}

   it prints=> First Sever created and started at port 3000
```
